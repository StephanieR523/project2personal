Project Description:

For Our project, we wanted to create a song search website using 2 different APS's.
The 2 API's we used for this project was LAST FM and YOUTUBE. LAST FM API will gave us data (TITLE, ALBUM NAME, and ARTIST NAME) and mp3 file that will lead to Last Fm website.
On the other hand Youtube Api will work with Last Fm API to gave us youtube video for the specific song that a user requested.

Search for the Song

Details: By clicking of the submit button our Last fm api will take place and it will gave us 5 songs with the Song Ttile, Artist Name, Album name and Link to Last Fm website.

Results

Details: After a user gets data from Last Fm API, a user can click on the row to see the Youtube results with Published Date, Views, Likes, Dislikes and a link to a youtube video.
